# FastAPI Hotel Room Booking System

This project is a complete backend application built using FastAPI.

## Features

- Room Management APIs
- Booking System
- Check-in / Checkout Workflow
- CRUD Operations
- Search, Sorting, Pagination
- Swagger API Testing

## Tech Stack

- FastAPI
- Python
- Pydantic

## How to Run

uvicorn main:app --reload
